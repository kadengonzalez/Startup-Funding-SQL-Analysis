# Startup Funding SQL Analysis (Crunchbase)

## Overview
Analyzed Crunchbase startup funding data using SQL to explore funding totals,
company status, and industry trends for venture investment evaluation.

## Skills & Tools
- SQL (SELECT, WHERE, GROUP BY, HAVING, ORDER BY)
- Aggregation and filtering
- Ranking and trend analysis

## Key Insights
- Identified industries with the highest funding activity
- Analyzed company funding levels by status
- Ranked startups based on total funding raised

## Repository Contents
- `queries.sql` – SQL queries used for analysis
- `milestone.pdf` – Completed project submission with results and explanations
